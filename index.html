Modal editing

  • Vim is a modal editor. That just means it has some different modes!
  • In Normal Mode, every key is a shortcut.
  • In Insert Mode, you can insert text like in any other text editor.
  • In Visual Mode, you can select text.
  • Normal Mode: every key is a shortcut
  • Insert Mode: insert text
  • Visual Mode: select text
  • Command Mode: enter commands (like saving)
